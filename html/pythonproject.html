Covid Robo

This is a python coding project that I and my brother did in a 2021 Febuary Hacakthon(Tinker for Change). We first didn't know what to do at all in the Hacakthon but then we thought of a perfect thing to do with python and that will also help everyone out. So, we created Covid Robo.

What is Covid Robo?

Covid Robo is basically a program that is build with python. It's meant to provide you important information about Covid-19. There are four options in the program. Background which provides you with some background info about Covid-19 and where it came from. Data which gives you real time data of all the cases around the world or also for a specific country. Quiz gives you a basic quiz about some important rules for Covid-19. Last option gives you More_info about Covid-19.
